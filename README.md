# Kasumi
> 一个简单的Typecho博客主题

## 主题特色
- Vue
- 轻量级
- 响应式
- 多语言
- 无评论

这是一个简单到没有评论系统的博客主题，支持多国语言翻译，部分组件使用Vue3渲染。
