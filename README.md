# Kasumi
> 一个简单的Typecho博客主题
> 与Bocchi主题可以无缝切换！

## 主题特色
- Vue
- 轻量级
- 响应式
- 多语言
- 无评论

这是一个简单到没有评论系统的博客主题，支持多国语言翻译，部分组件使用Vue3渲染。

## 短代码使用
> 更多待更新

| :短代码: | :说明:                                  |
| ------ | ------------------------------------- |
| 提示   | [t]文字[/t]                           |
| 按钮   | [b url="链接"]文字[/b]                |
| 纸片   | [c url="链接" img="图片链接"]文字[/c] |
